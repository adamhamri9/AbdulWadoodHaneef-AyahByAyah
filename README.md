**Audio file naming convention:**

Each audio file is named as `00X00X.mp3`, where:

* The **first three digits** represent the **surah number**.
* The **last three digits** represent the **ayah number**.

| File Name   | Surah | Ayah |
|------------|-------|------|
| 002003.mp3 | 2     | 3    |
| 002004.mp3 | 2     | 4    |
